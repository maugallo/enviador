¿Qué es esto?

Esta página es uno de mis primeros proyectos utilizando Django. Este sitio permitirá al usuario poder enviar un email a cualquier persona utilizando el correo electrónico que la propia página ofrece.

Se puede comenzar haciendo click acá.

¿Qué aprendí con este proyecto?

Con este proyecto aprendí a:

  • Poder enviar emails a través de Django.
  • Conceptos internos sobre cómo una página web envía emails, implementando los servidores SMTP o el protocolo de seguridad TLS.
  • Crear un form básico donde poder ingresar información y verificarla.
  • Conectar una function-based view con un form.
  • Permitir adjuntar archivos al mail.
  • Usar la librería Environ para proteger información sensible.
